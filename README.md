# JustShowMe-releases

Public release artifacts and auto-update feed for JustShowMe Desktop. Source lives in the private JustShowMe repo.

## Getting help

Where you report something depends on what it is:

| Type of report | Where to go |
| --- | --- |
| 🐛 Bug, crash, or something not working | [Open a bug report](https://github.com/j-shiels/JustShowMe-releases/issues/new?template=bug_report.yml) |
| 💡 Feature request or idea | [Open a feature request](https://github.com/j-shiels/JustShowMe-releases/issues/new?template=feature_request.yml) |
| 🔒 Security / vulnerability | **Privately** — see [SECURITY.md](SECURITY.md). Do **not** open a public issue. |
| 💳 Account, billing, purchases, licensing | Email **support@robn.studio** |
| 🛟 Anything sensitive or private | Email **support@robn.studio** or the [support page](https://robn.studio/justshowme/support) |

> ⚠️ **Heads up:** JustShowMe captures screen recordings, screenshots, and annotations.
> Issues here are **public** — redact anything private (screen contents, file names,
> account details) before attaching screenshots or logs.
