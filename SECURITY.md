# Security Policy

## Reporting a vulnerability

**Please do not report security vulnerabilities through public GitHub issues, discussions, or pull requests.**

If you believe you've found a security vulnerability in JustShowMe Desktop, report it
privately by emailing:

**support@robn.studio**

Please include as much of the following as you can:

- A description of the vulnerability and its potential impact
- Steps to reproduce, or a proof of concept
- The JustShowMe version and operating system affected
- Any relevant logs (redact anything personal)

You'll get an acknowledgement as soon as possible. Please give a reasonable window to
investigate and ship a fix before any public disclosure.

## Scope

This repository hosts the public release artifacts and auto-update feed for JustShowMe
Desktop. The application source lives in a private repository — reports about the app
itself are still welcome at the address above.
